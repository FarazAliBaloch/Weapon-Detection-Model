Weapon Detection Using YOLOv8

This AI-powered web app detects weapons in images, video, and webcam feeds using a custom-trained YOLOv8 model. It helps in enhancing public safety by identifying potentially harmful objects in real-time.

How It Works

  • YOLOv11: You Only Look Once (v8) is a real-time object detection algorithm that predicts bounding boxes and class probabilities in one evaluation.
  • Custom Model: Our team trained YOLOv11 on a specialized dataset of weapon images for precise detection.
